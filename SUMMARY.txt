================================================================================
                     TROUBLESHOOTING
================================================================================

Problem: Module not found
Solution: source venv/bin/activate && pip install -r requirements.txt

Problem: GUI won't start
Solution: brew install python-tk  (Mac)
          sudo apt-get install python3-tk  (Ubuntu)

Problem: No data extracted
Solution: Check FIDE ID is correct, verify internet connection

Problem: Export fails
Solution: Close the file if it's already open in Excel
